The Problem
"We make 35,000 decisions a day — but how many do we learn from?"

❌ No Structure

Decisions made without documentation or context

❌ No Feedback

No system to track what actually happened

❌ No Learning

Same mistakes repeated because patterns are invisible

The Solution

An operating system for your life decisions

📝 Log

Decisions with context, alternatives & expected outcomes

📊 Track

Schedule check-ins to record actual results over time

🧠 Advise

AI advisor that knows your personal decision history

🔍 Discover

Auto-detect behavioral patterns and blind spots

"Decide better. Learn faster. Live intentionally."

Core Features

What LifeOS Does

📓 Decision Journal

Structured logging with categories, alternatives, confidence levels, and expected metrics

🔔 Outcome Tracking

Scheduled check-ins to measure satisfaction, record reflections, and track lessons learned

💬 AI Advisor

Conversational AI with SSE streaming that knows your full decision history

🔬 Pattern Detection

Auto-discovers timing effects, overconfidence bias, and category-specific tendencies

📈 Analytics

Visualize decision quality, satisfaction trends, and category breakdowns over time

🧠 Memory System

AI extracts key facts from your reflections for personalized long-term guidance

AI Context Pipeline

Every AI response is personalized through a 4-layer context assembly

1
User Profile ~150 tokens
Pre-computed longitudinal behavioral summary
↓
2
Category Context ~100 tokens
Aggregated stats for the specific decision category
↓
3
RAG — Similar Decisions ~500 tokens
Vector search finds semantically similar past decisions
↓
4
Behavioral Patterns ~100 tokens
Auto-detected tendencies (timing, confidence calibration)
↓
+
User Memories ~100 tokens
Facts extracted from past reflections & conversations

Security

Authentication & Security

LayerImplementation
Passwordsbcrypt hashing with salt
Access TokensJWT — 15 min expiry
Refresh TokensSecure random — 7 day expiry
Mobile Storageexpo-secure-store (encrypted keychain)
API ProtectionBearer token middleware on all routes
Auto-RefreshAxios interceptor: 401 → refresh → retry
Data SafetySoft deletes — never hard-delete user data

Auth Flow

User → Login (email/pass) → Server validates → bcrypt compare → Issue accessToken + refreshToken → Store in SecureStore → Attach to every API request → 401? Auto-refresh → retry
